# Trader Behavior Analysis using Market Sentiment

This project analyzes trader performance and behavior under different Bitcoin market sentiment conditions such as Fear and Greed.

## Files Included
- notebook_1.ipynb: Complete analysis done in Google Colab
- csv_files/: Processed datasets
- outputs/: Visualizations generated from analysis
- ds_report.pdf: Summary report of insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

All analysis was performed in Google Colab as per assignment instructions.
